# Пакет компонента Log входит в состав GM Framework.

Пакет компонента Log  предназначен для ведения журнала общего назначения. Он поддерживает несколько бэкэндов журналов, форматирование сообщений, отправляемых в журнал, и фильтрацию сообщений при записи вы журнал.

## Ресурсы
- [GM Framework](https://gearmagic.ru/framewor/k)
- [Документация](https://gearmagic.ru/package/framework-log/)

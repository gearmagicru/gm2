# Пакет компонентов Db входит в состав GM Framework.

Пакеты компонентов абстрагирует доступ к базе данных с помощью объектно-ориентированного API для построения запросов. Компонент использует разные адаптеры хранения для доступа к различным поставщикам баз данных, таким как MySQL, PostgreSQL, Oracle, IBM DB2, Сервер Microsoft Sql, PDO и т.д.

## Ресурсы
- [GM Framework](https://gearmagic.ru/framework/)
- [Документация](https://gearmagic.ru/package/framework-db/)

# Пакет компонентов API входит в состав GM Framework.

Пакет компонентов API помогает каждому модулю или его расширениям обрабратывать запросы клиентов через свой API-интерфейс.

## Ресурсы
- [GM Framework](https://gearmagic.ru/framework/)
- [Документация](https://gearmagic.ru/package/framework-api/)

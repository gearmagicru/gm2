# Пакет компонентов Stdlib входит в состав GM Framework.

Gm\Stdlib - это набор компонентов, реализующий базовый класс общего назначения для различных областей, например:
- коллекции;
- базовые объекты наследования;
- обертки и т.п.

## Ресурсы
- [GM Framework](https://gearmagic.ru/framework/)
- [Документация](https://gearmagic.ru/package/framework-stdlib/)

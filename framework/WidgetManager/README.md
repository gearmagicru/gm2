# Пакет компонента WidgetManager входит в состав GM Framework.

Пакет компонента включает "Менеджер виджетов", предназначен для управления установленными виджетами на страницах сайта.

## Ресурсы
- [GM Framework](https://gearmagic.ru/framework/)
- [Документация](https://gearmagic.ru/package/framework-widgetmanager/)

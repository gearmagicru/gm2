# Пакет компонента Filesystem входит в состав GM Framework.

Пакет компонента содержит простые в использовании адаптеры для работы с локальными файловыми системами: Amazon S3, Rackspace Cloud Storage, FTP и др.

## Ресурсы
- [GM Framework](https://gearmagic.ru/framework/)
- [Документация](https://gearmagic.ru/package/framework-filesystem/)
